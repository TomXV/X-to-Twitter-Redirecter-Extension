X to Twitter Redirecter

Twitter.comを取り戻そう！

長年使い慣れたTwitter。最近、突如としてドメインがtwitter.comからx.comに変更されてしまいました。

「そうはさせるか！」

そう思った方々に朗報です。この拡張機能を使えば、従来のTwitterドメインを維持し続けることができます！

特徴

  • x.comをtwitter.comに自動的にリダイレクト
  • 使い慣れたURLでTwitterを楽しめる
  • シンプルで軽量な拡張機能

今すぐ試してみよう！

現在、先行リリース中です。気に入るかどうかわかりませんが、一度入れてみてお試しあれ。へへへ...楽しみにしてますぜ！
